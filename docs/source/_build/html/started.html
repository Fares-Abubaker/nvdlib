Getting started¶

Installation¶

To use NVDLib, first install it using pip:

(.venv) $ pip install nvdlib

Requirements¶

Next, download the Requests library for NVDLib to function. Currently this is the only library required.

(.venv) $ pip install requests

Importing NVDLib¶

Before you begin utilizing NVDLib make sure you import the nvdlib.py module:

import nvdlib
